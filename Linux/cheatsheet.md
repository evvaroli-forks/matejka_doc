---
title: Cheat Sheet der Linux Befehle
---



**root** Rechte erlangen

```sh
$ sudo -i
```



**Tastaturlayout** einstellen (Kali, Ubuntu, ...)

```sh
$ setxkbmap -layout de
```

Metasploitable:

```sh
$ loadkeys de
```



**Shutdown**

```sh
$ shutdown -h now
```



IP Adresse

```sh
$ ifconfig [eth0]
$ ip a
```



Default Gateway

```sh
$ route -n
```
